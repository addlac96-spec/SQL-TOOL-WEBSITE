# Changelog

All notable changes to the SQL Teacher Tool will be documented in this file.

## [v2.4.0] - 2026-02-19
### Improved
- **Intro Lesson Cards**: Renamed "Tutorial Intro" → "Welcome to SQL" and "Intermediate SQL Intro" → "Leveling Up" with action-oriented descriptions.
- **Study Guide Content**: Rewrote Lesson 101 guide syntax, examples, and cheatsheet to teach `SELECT * FROM` instead of generic `Hello World`.
- **Quick Learn Panel**: Updated tutorial syntax/example for Lesson 101 to match the lesson's actual tasks.

### Changed
- **Docs Structure**: Moved `ARCHITECTURE.md` and `CONTENT_GUIDE.md` into `docs/` folder. Added documentation index to `README.md`.

## [v2.3.0] - 2026-02-19
### Added
- **Submit vs Run**: Separated query execution from answer validation. "Run Query" allows free exploration; "Submit Answer" validates the result.
- **Resource Hub Integration**: Added "Read Guide" button to every lesson, linking directly to the comprehensive study guides.
- **UI Cleanup**: Removed redundant result status notifications to streamline the feedback loop.

## [v2.2.0] - 2026-02-19
### Added
- **Refined Conceptual Hints**: Completed the 100% conceptual rewrite of all 500+ hints across Ecommerce, SaaS, HR, Healthcare, and Finance.
- **Verification Portfolio**: Added detailed browser verification recordings and screenshots for the new hint system.

## [v2.1.0] - 2026-02-18
### Added
- **Beginner Advanced Tracks**: Introduced Lesson 201 "Master Challenges" for all industries.
- **Random Drill Engine**: Updated the lesson logic to pull random question variants instead of fixed steps.
- **Table Data Previews**: Added the ability to see 5-row previews of all available tables in the sidebar.

## [v2.0.0] - 2026-02-17
### Added
- **Massive Content Expansion**: Scaled from ~50 questions to over 450 unique manager requests.
- **Industry Selection**: Added the Industry Switcher for distinct learning contexts.

## [v1.0.0] - 2026-02-15
### Added
- **Initial MVP**: Basic SQL editor, SQL.js integration, and the first set of E-commerce lessons.
- **Dark Mode UI**: Professional aesthetic baseline.
